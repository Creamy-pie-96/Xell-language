# Fonts Directory

Place a monospace `.ttf` font file here named `JetBrainsMono-Regular.ttf`.

The build script (`build.sh`) will automatically download JetBrains Mono if it's
not present. You can also manually download it from:

https://github.com/JetBrains/JetBrainsMono/releases
